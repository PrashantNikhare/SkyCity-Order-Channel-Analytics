# 🍽️ SkyCity Auckland Restaurants & Bars
## Order Channel Performance and Market Share Analytics

An interactive Data Science project that analyzes restaurant order channel performance across multiple platforms such as In-Store, Uber Eats, DoorDash, and Self Delivery. The project provides business insights through Exploratory Data Analysis (EDA) and an interactive Streamlit dashboard.

# 📌 Project Overview

The Auckland hospitality industry has experienced rapid growth in online food ordering through delivery aggregators. This project analyzes restaurant performance across different order channels to understand customer behavior, market share, regional preferences, cuisine trends, and dependency on third-party delivery platforms.

The dashboard enables restaurant owners and analysts to make data-driven business decisions using interactive filters and visualizations.

---

# 🎯 Project Objectives

- Analyze total order volume by channel
- Calculate market share for each order channel
- Compare delivery versus in-store performance
- Analyze ordering behavior across Auckland subregions
- Compare cuisine-wise and segment-wise performance
- Detect restaurants with high aggregator dependency
- Build an interactive business dashboard using Streamlit

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/PrashantNikhare/SkyCity-Order-Channel-Analytics.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
